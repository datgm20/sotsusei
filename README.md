#### Trelloの準備
- チーム制作をする際に、仕様や設定、作業リスト、作業担当、締め切りなどを管理するためのサービス
- https://trello.com/ でGmailでアカウント作成
  - 新規のワークスペース名に、まずは自分専用のワークスペース名を自分で考えて入力
  - ワークスペースのタイプは自由に。その他など
  - プランの選択画面では、一番下に灰色で無料で利用するボタンがあるのでそれをクリック
  - グループ用ワークスペースと卒業制作用のボードの作成
    - じゃんけんなどで作業担当者を決めて、一人がワークスペースを非公開で作成(DAT20_GroupA1など)
    - タイプは「エンジニアリング/IT」
    - 招待のところにメンバーのメールアドレス、あるいは、アカウントを入力(メールアドレスだと表示されない場合があるので、その場合はアカウント)
    - ボードをワークスペースに公開で作成
　　- ボードの作成(Info / Backlog / TODO / Check / Done)
  - Info
    - 企画の参考資料やゲーム画面サイズ、ストーリー、参考URL、使用アセット、メンバー情報などのカードを作成
  - Backlog
    - 思いついた作業のカードをこのリストで作成
  - TODO
    - 今後の1～2週間で実施予定の作業リスト
    - 週初めの打ち合わせで作業することになったものをBacklogからTODOに移して、期限と担当者を決める
  - Check
    - 作業が完了したらTODOからこのリストに移す
    - 打ち合わせの時などにここに移された項目を確認して、グラフィックをプロジェクトに組み込んだり、できあがった動きをチーム内で確認したりする
    - 問題点や修正が必要になったら、Backlogに戻すか、期限を更新してTODOに戻す
  - Done
    - Checkが完了したらこのリストに移して作業完了となる
    - チェックすることがない作業は、直接このリストに動かしてもよい
- Slackは連絡。Trelloは作業に関する情報の整理と共有
